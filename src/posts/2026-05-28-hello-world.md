# ブログをはじめました

ポートフォリオサイトに、記事を書き溜めていく Blog セクションを追加しました。
このページがその最初のテスト記事です。

## 仕組み

記事は `src/posts/` フォルダの中に Markdown ファイルとして置いてあります。
ファイル名の規則は `YYYY-MM-DD-<slug>.md` で、frontmatter にタイトル・日付・タグ・概要を書くだけ。

```js
// src/lib/posts.js で全ファイルを取得
const modules = import.meta.glob('../posts/*.md', {
  eager: true,
  query: '?raw',
  import: 'default',
})
```

Vite の `import.meta.glob` を使って **ビルド時に全 md を一括読み込み**しているので、
記事を追加して `npm run deploy` するだけで反映されます。
ライブラリの再起動や DB 更新は不要です。

## 何を書く？

技術メモが中心になる予定です。

- React / TypeScript で気づいたこと
- Claude や AI ツールの活用記録
- 個人開発で詰まった話とその解決

たまに **Life** タグの雑記も書くかもしれません。
